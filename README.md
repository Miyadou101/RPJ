# RPJ  
*A simple text-based RPG made with Java.*

---
## Planned Features

- Core combat and damage types (physical/magical)
- Inventory, equipment, and durability (melee, ranged, armor)
- Procedural loot and rarity
- Quests and encounters
- Save/load system
- Settings (audio, input, difficulty)
- UI/UX polish and accessibility
- Localization
- Automated tests and CI

### Characters (planned)

- Classes and primary attributes:
  - Warrior → Strength (STR)
  - Rogue → Agility (AGI)
  - Wizard → Intelligence (INT)

- Level-up rules:
  - Each level grants +1 to the class’s primary attribute by default.
  - Starting at level 3, the player may choose to allocate a level-up to a non-primary attribute instead.

- Attribute caps (per character):
  - Main attribute may reach a cap of 100 (the primary).
  - A Secondary is  capped at 40.
  - The Rest are capped at 20

- Balance goal: Encourage class identity while allowing limited build variation without overpowering hybrids.

---

##  Description  
RPJ is a simple text-based role-playing game written in **Java**, featuring multiple character and weapon classes. Players can explore different playstyles by choosing between melee, ranged, and magic combat types.
